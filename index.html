💡 Lights Out Game

🎮 Game Rules

  • Click a square to toggle it and its neighbors.
  • Turn all squares white or black to win.
  • Click "Reset" for a new puzzle.
  • "Show Status" gives a progress hint.

📝 Addendum

  • Started with a basic 5x5 grid and styled using CSS Grid.
  • Used event listeners and DOM manipulation to build interactivity.
  • Ensured solvable puzzles by simulating clicks from a blank board.
  • Victory condition checks if all tiles are white or all are black.
  • Faced challenges in randomization and solved them using toggle logic.
  • This project deepened my understanding of game logic and UI design.
